# WebSocket Chat App

## Description

This is a simple WebSocket-based chat application created with React, TypeScript, and WebSocket server (using the `ws` library). The goal of this project is to practice working with WebSockets, React, and real-time communication.

## Features

### WebSocket Real-Time Communication:

- Messages are sent and received in real-time.
- Server broadcasts the messages to all connected clients except the sender.

## Tech Stack

### Frontend:
- React (with TypeScript)
- Tailwind CSS
- Vite (for build tool)

### Backend:
- Node.js
- WebSocket Server (ws library)
